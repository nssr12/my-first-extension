# My First Extension

إضافة لمتصفح Chrome للتحكم بالفيديوهات في المواقع المختلفة مثل YouTube و Kick وغيرها.

## المميزات

- ريماب عام للاختصارات على جميع المواقع التي تحتوي على فيديو
- تقسيم الفيديو إلى شبكة `3x3`
- ربط كل مربع بإجراء مختلف عند استخدام عجلة الماوس
- دعم اختصارات لوحة المفاتيح والماوس
- إظهار حالة الصفحة الحالية من داخل نافذة الإضافة
- تفعيل يدوي إذا لم تعمل الإضافة تلقائيًا على بعض الصفحات
- منع مواقع معينة من استخدام الإضافة
- مؤشر رقم الصوت أعلى يسار الفيديو

## أهم الأوامر المدعومة

- `ACTION:TOGGLE_PLAY`
- `ACTION:TOGGLE_FULLSCREEN`
- `ACTION:TOGGLE_MUTE`
- `ACTION:VOLUME:+4`
- `ACTION:VOLUME:-4`
- `ACTION:SEEK:+1`
- `ACTION:SEEK:-1`
- `ACTION:SEEK:+5`
- `ACTION:SEEK:-5`
- `ACTION:SEEK:+10`
- `ACTION:SEEK:-10`
- `ACTION:SEEK:+30`
- `ACTION:SEEK:-30`
- `ACTION:SPEED:+0.25`
- `ACTION:SPEED:-0.25`
- `ACTION:TOGGLE_PIP`

## طريقة التثبيت في Chrome

1. افتح `chrome://extensions`
2. فعّل `Developer mode`
3. اضغط `Load unpacked`
4. اختر مجلد المشروع

## طريقة الاستخدام

### من نافذة الإضافة

- تفعيل أو إيقاف الريماب العام
- منع الموقع الحالي من استخدام الإضافة
- تعديل مدة ظهور الـ Overlay
- إضافة قواعد `From -> To`
- اختيار الأكشن من القائمة الجاهزة بدل النسخ واللصق
- فحص حالة الصفحة الحالية
- التفعيل اليدوي إذا لم تعمل الإضافة مباشرة

### من صفحة الإعدادات

- تعديل شبكة `Interaction Zones (3×3)`
- اختيار أوامر `Wheel Up / Wheel Down` لكل مربع
- مشاهدة قائمة المواقع الممنوعة
- تعديل لون وحجم مؤشر رقم الصوت

## الملفات المهمة

- [manifest.json](./manifest.json): إعدادات الإضافة وصلاحياتها
- [popup.html](./popup.html): واجهة نافذة الإضافة
- [popup.js](./popup.js): منطق نافذة الإضافة
- [options.html](./options.html): صفحة الإعدادات الداخلية
- [options.js](./options.js): منطق صفحة الإعدادات
- [content.js](./content.js): المنطق الذي يعمل داخل صفحات المواقع

## ملاحظات

- بعض الصفحات المحمية من المتصفح مثل صفحات `chrome://` لا يمكن تشغيل الإضافة عليها
- إذا لم تعمل الإضافة على صفحة معينة، افتح نافذة الإضافة واضغط `فحص الصفحة` ثم `تفعيل يدوي`

## التطوير لاحقًا

- إضافة تصنيفات للأكشنات داخل الـ popup
- دعم واجهة عربية كاملة داخل صفحة الإعدادات
- إضافة إعدادات أكثر لمؤشرات الفيديو مثل مكان مؤشر الصوت وخلفيته
