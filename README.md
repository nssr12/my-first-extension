# Video Interaction Zones

إضافة Chrome للتحكم بالفيديوهات في أغلب المواقع من خلال اختصارات لوحة المفاتيح والماوس، مع شبكة تفاعلية `3×3` فوق الفيديو نفسه.

## الفكرة

الإضافة لا تعمل على الصفحة كلها بشكل عشوائي، بل تحاول تنفيذ الأوامر فقط عندما يكون الماوس فوق الفيديو. هذا يجعل الاختصارات مناسبة لمواقع مثل YouTube وKick والمشغلات المدمجة داخل مواقع أخرى.

## المميزات

- ريماب عام للاختصارات على جميع المواقع التي تحتوي على فيديو
- دعم اختصارات الكيبورد والماوس
- شبكة `Interaction Zones (3×3)` مع أوامر مستقلة لكل مربع
- Overlay داخل الفيديو لعرض التلميحات
- مؤشر رقم الصوت أعلى يسار الفيديو
- قائمة أكشنات جاهزة داخل الـ popup
- منع مواقع محددة من استخدام الإضافة
- فحص حالة الصفحة الحالية من داخل نافذة الإضافة
- تفعيل يدوي إذا لم يتم حقن الإضافة تلقائيًا

## الأكشنات المدعومة

- `ACTION:TOGGLE_PLAY`
- `ACTION:TOGGLE_FULLSCREEN`
- `ACTION:TOGGLE_MUTE`
- `ACTION:TOGGLE_PIP`
- `ACTION:SEEK:+1`
- `ACTION:SEEK:-1`
- `ACTION:SEEK:+5`
- `ACTION:SEEK:-5`
- `ACTION:SEEK:+10`
- `ACTION:SEEK:-10`
- `ACTION:SEEK:+30`
- `ACTION:SEEK:-30`
- `ACTION:VOLUME:+4`
- `ACTION:VOLUME:-4`
- `ACTION:SPEED:+0.25`
- `ACTION:SPEED:-0.25`

## التثبيت

1. افتح `chrome://extensions`
2. فعّل `Developer mode`
3. اضغط `Load unpacked`
4. اختر مجلد المشروع

## الاستخدام

### من نافذة الإضافة

- تشغيل أو إيقاف الإضافة بشكل عام
- منع الموقع الحالي
- تعديل مدة ظهور الـ Overlay
- إضافة قاعدة `From -> To`
- اختيار أكشن جاهز من القائمة
- فحص حالة الصفحة
- التفعيل اليدوي عند الحاجة

### من صفحة الإعدادات

- تعديل أوامر شبكة `3×3`
- مشاهدة شرح الأكشنات
- إدارة المواقع الممنوعة
- تعديل لون وحجم مؤشر رقم الصوت

## الملفات الأساسية

- `manifest.json`: تعريف الإضافة والصلاحيات
- `content.js`: منطق التحكم بالفيديو داخل المواقع
- `popup.html` و `popup.js`: نافذة الإضافة
- `options.html` و `options.js` و `options.css`: صفحة الإعدادات الداخلية

## ملاحظات مهمة

- بعض الصفحات المحمية مثل `chrome://` لا تسمح Chrome بتشغيل إضافات المحتوى عليها
- بعض المواقع تستخدم مشغلات خاصة أو `iframe` وقد تحتاج إلى `تفعيل يدوي`
- إذا ظهرت حالة أن الإضافة شغالة لكن الأمر لا ينفذ، فغالبًا المشكلة تكون في اكتشاف الفيديو أو في طبقات المشغل فوقه
